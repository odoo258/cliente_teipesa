Fiscal Sequences

Easy Setup

1. Create fiscal sequences in Accounting>>Settings>>Tax Configuration>>Fiscal Numbers

2. Fill required fields, in "Document Type" select 'Remission'. Save

3. In Stock Transactions select "Fiscal Control".

- When activate the Fiscal Control, select your sequence.
